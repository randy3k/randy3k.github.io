R package(s)

iterpc is an R package helps in generating (iteratively) permutations and combinations with or without replacement; with distinct items or non-distinct items (multiset); calculating number of combinations or permutations in various situations. All generated sequences are in lexicographical order (AKA, dictionary order). The algorithms to generate permutations and combinations are memory efficient, only previous sequence is required to generate next sequence. These iteratively algorithms enable users to process all sequences without putting all results in the memory at the same time. The algorithms are written in C/C++ and hence very computational efficient.

RSpace

I am working on a naive cocoa GUI for R. Many existing GUIs such as RStudio or RGUI are running R in the main thread. I am implementing it in a different approach. In RSpace, R is running in a different thread than the main thread where the GUI runs. It will result in a less laggy GUI. Many features are still unavailable.

Sublime Text 2/3

AlignTab is a regex alignment plugin which helps Sublime users to align their codes. This plugin is inspired by the Vim plugin tabular.
ChangeList allows user to navigate between edit points, similar to what :changes does in VIM. It is now rewritten for ST3 compatibility.
AutoWrap wraps line, and is particular useful for latex.
Enhanced-R is a comprehensive set of tools for helping writting R codes in ST. Seemingly the first R plugin which supports multi-platform.
LaTeXTab turns excel or cdv tables into beautiful latex tables.
LaTeXSQ is a LaTeX plugin for ST.
