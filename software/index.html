R package(s)

iterpc helps in 1) generating permutations and combinations with or without replacement; 2) with distinct items or non-distinct items (multiset); 3) calculating number of combinations or permutations in various situations. All generated sequences are in lexicographical order (AKA, dictionary order). The algorithms to generate permutations and combinations are memory efficient. These iterative algorithms enable users to process all sequences without putting all results in the memory at the same time. The algorithms are written in C/C++ for faster performances.

Sublime Text 2/3

AlignTab is a regex alignment plugin which helps Sublime users to align their codes. This plugin is inspired by the Vim plugin tabular.
ChangeList allows user to navigate between edit points, similar to what :changes does in VIM. It is now rewritten for ST3 compatibility.
AutoWrap wraps line, and is particular useful for latex.
R-Box is a comprehensive set of tools for helping writting R codes in ST.
LaTeXTab turns excel or cdv tables into beautiful latex tables.
LaTeX-Plus is a LaTeX plugin for ST.
SyntaxMgr is a plugin manages syntax settings.
Project-Manager is a plugin manages Sublime Projects.
UnitTesting is a unittest framework for ST2/3 based on unittest module in python.
Whitespace trims whitespaces automatically.
